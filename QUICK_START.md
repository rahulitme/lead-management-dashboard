# Lead Management Dashboard

A full-stack CRM-style application built with React + Node.js + MongoDB.

## Quick Start

### Setup Backend
```bash
cd backend
npm install
cp .env.example .env
# Update .env with MongoDB connection string
npm run seed  # Seeds 500 dummy leads
npm start     # Starts on localhost:5000
```

### Setup Frontend
```bash
cd frontend
npm install
cp .env.example .env
npm run dev   # Starts on localhost:3000
```
